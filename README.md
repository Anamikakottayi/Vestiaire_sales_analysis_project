# 🛍️ E-Commerce Sales Analysis (SQL + Power BI)

## 📊 Project Overview
This project analyzes an e-commerce dataset using SQL and Power BI to generate business insights.

## 🛠 Tools Used
- SQL (MySQL)
- Power BI
- Python (Data Cleaning)

## 📌 Key Insights
- Women’s products dominate the platform
- Top brands drive majority of sales
- Faster shipping leads to higher conversions
